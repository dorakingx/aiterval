# Contributing

Use Node 20.19+ and pnpm 10. Run `pnpm install`, create a focused branch, and keep changes local-first. New wait adapters must use semantic state signals, include fixtures, and never inspect conversation content. New exercises must be original and pass runtime validation.

Before opening a pull request, run `pnpm check`. Describe user-visible behavior, privacy impact, tested browsers, and any new permissions. Do not commit build caches, credentials, analytics, remote code, or copied learning material.
