# Product specification

## Problem

Busy professionals use AI throughout the workday but struggle to create a separate English-learning habit. AI generation already creates short, repeated pauses.

## Promise

Convert one waiting opportunity into one useful English listen without trapping the user in a learning product.

## MVP success

A small success is listening once, answering once, or reviewing one missed expression. Primary progress is actual active sprint duration and opportunities converted per week. A comeback after inactivity begins with: “Welcome back — listening once is enough today.”

## Primary flow

1. User submits a prompt on a supported AI site.
2. Generation-state detection survives the configured threshold.
3. One listening sprint appears in a non-blocking corner overlay.
4. The transcript remains hidden during audio.
5. User answers one question and receives evidence-grounded feedback.
6. If the AI finishes, a readiness notice appears while the current sentence
   continues; the user chooses return, finish, or save.
7. Progress is stored locally immediately.

## Content and personalization

The MVP contains 132 original exercises across academic lectures, research, summer school, networking, discussions, daily conversation, AI/software, quantum computing, machine learning, Web3, and names/numbers. Five question types and five difficulty levels are represented. Recommendations remain local and transparent.

## Non-goals

Accounts, cloud sync, remote AI generation, microphone scoring, an infinite feed, punitive daily streaks, ads, telemetry, aggressive notifications, and automatic Chrome Web Store publishing are excluded.
