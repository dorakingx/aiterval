## Submission lead

At an international summer school, I struggled to follow English lectures but
had little dedicated study time. I was also repeatedly waiting for AI tools.
AIterval turns those 15–90 second waits into one short listening sprint.

The submitted local-first extension uses 132 original pre-authored exercises.
It requires no OpenAI API key, stores progress locally, observes generation-state
controls only, and never reads prompts or responses. Codex with GPT-5.6 was used
honestly during architecture, implementation, testing, debugging, and refinement;
runtime AI exercise generation is not part of the submitted product.
